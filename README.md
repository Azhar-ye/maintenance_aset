# Sistem Informasi Monitoring Maintenance dan Cost Tracking Aset

## Deskripsi
Project ini merupakan aplikasi berbasis web yang dibangun menggunakan Framework CodeIgniter 4 untuk membantu proses pengelolaan aset perusahaan.

## Fitur
- Login Multi Role
- Dashboard
- Data Aset
- Kategori
- Lokasi
- Vendor
- User Management
- Laporan Kerusakan
- Jadwal Maintenance
- Maintenance
- Cost Tracking
- QR Code
- Export PDF
- Export Excel
- Activity Log
- Profil Pengguna

## Teknologi

- PHP 8
- CodeIgniter 4
- Bootstrap 5
- MySQL
- JavaScript
